Ementa da Disciplina

Linguagem de marcação de hypertexto - html. Linguagem de marcação e estilos - css. Linguagem Javascript. Programação de páginas dinâmicas com ajax e php. Criação de um crud.

Objetivos

  1. Estruturar páginas web, utilizando a linguagem de marcação de texto(html 5),para a formação de um arcabouço sobre o qual serão construidas funcionalidades dinâmicas.
  2. Aplicar caracteristicas de estilo a páginas WEB, utlizando a linguagem de marcação de estilos (CSS 3), para praticar técnicas de engenharia de software como facilidade de compreensão, reutilização de código, manutenibilidade e interoperabilidade.
  3. Empregar programabilidade em páginas web, utilizando linguagem javascript, mais usada no mercado, para o desenvolvimento de um sistema web com funcionalidades dinâmicas.
  4. empregar programabilidade em páginas web, utilizando linguagem AJAX e PHP, bastante comun em sistemas legado, para o desenvolvimento de um sistema web com funcionalidades dinâmicas.
  5. Integrar um SGBD em sistema web, baseando-se na linguagem PHP e na classe PDO, para que o sistema seja capaz de lidar com massas de dados estrututrados.
